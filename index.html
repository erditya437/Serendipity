Tujuan saya adalah untuk membuat alat digital yang sederhana namun bermakna untuk mengungkapkan.

Cara Menggunakan

1. Isi Detail

Masukkan nama pasangan, pilih foto, dan tuliskan catatan pribadi untuk setiap foto.

2. Sesuaikan

Pilih tema warna dan GIF utama yang paling sesuai dengan kepribadian Anda dan pasangan.

3. Bagikan

Generate pesan dan bagikan tautannya ke pasangan Anda melalui WhatsApp atau media sosial lainnya.

Tanya Jawab

Apakah aplikasi ini gratis?
Ya, aplikasi ini 100% gratis dan akan selalu begitu.
Apakah pesan saya aman?
Data pesan Anda hanya disimpan di browser Anda sendiri (Local Storage) dan tidak dikirim ke server mana pun. Ini berarti pesan Anda aman dari serangan SQL Injection dan XSS (Cross-Site Scripting). Tidak ada risiko CSRF (Cross-Site Request Forgery) karena tidak ada permintaan yang dikirim ke server. Singkatnya, pesan Anda hanya milik Anda.
